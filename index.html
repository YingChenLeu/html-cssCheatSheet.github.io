HTML & CSS Cheat Sheet

HTML Cheat Sheet

Concept Tag Purpose
Paragraph Tag p To present information in a paragraph.
Heading Tag h1, h2, h3, h4, h5, h6 To define headings with different levels of importance (h1 is most important).
Anchor Tag a To create hyperlinks that navigate to other pages or resources.
Image Tag img To display images on a webpage.
Ordered List ol To create a list with numbered items.
Unordered List ul To create a list with bullet points.
List Item li To define each item in an ordered or unordered list.
Table Tag table To create a table to organize data in rows and columns.
Table Row tr To define a row in a table.
Table Data td To define a cell in a table row (data cell).
Table Header th To define a header cell in a table row.
Div Tag div To group and style block-level elements.
Span Tag span To group and style inline elements.
Form Tag form To create a form for user input.
Input Tag input To define an input field in a form (e.g., text, password, checkbox).
Button Tag button To create a clickable button.
Label Tag label To define a label for form elements.
HTML Tag html To define the root element of an HTML document.
Body Tag body To define the body content of an HTML document.
Head Tag head To contain metadata and links to stylesheets, scripts, etc.
Title Tag title To define the title of the document, shown in the browser tab.


CSS Cheat Sheet

Concept Property Purpose
Color color Sets the color of the text.
Background Color background-color Sets the background color of an element.
Font Size font-size Defines the size of the font.
Font Family font-family Specifies the font to be used for text.
Text Alignment text-align Aligns text horizontally (e.g., left, center, right).
Margin margin Creates space around an element (outside its border).
Padding padding Creates space between an element's content and its border.
Border border Defines the border around an element (color, width, style).
Width width Sets the width of an element.
Height height Sets the height of an element.
Display display Specifies how an element is displayed (e.g., block, inline, flex).
Overflow overflow Controls what happens when content overflows its container (visible, hidden, scroll, auto).
Opacity opacity Sets the transparency level of an element.
Font Weight font-weight Defines the boldness of text (normal, bold, lighter, bolder, or numeric values).
Border Radius border-radius Rounds the corners of an element's border.
Background Image background-image Sets an image as the background of an element.
